Api rest service

REST API service for third-party developers has 2 resources:
1) Departments:

1.1. access to the collection of the departments of the company:

base-url/api/departments/ - method get

1.2. access to the department by its uuid:

base-url/api/departments/uuid - methods get, post, put, delete

2) Employees:

2.1. access to the collection of the employees of the company:

base-url/api/employees/ - method get

2.2. access to the employee by its uuid:

base-url/api/employees/uuid - methods get, post, put, delete
